# Инструкция по импорту коллекции Postman

Для того, чтобы импортировать себе коллекцию Postman нужна ссылка. 

Ссылка на импорт коллекции: https://api.postman.com/collections/37103417-baf0815a-25a3-4f34-b1ec-062ec180371e?access_key=(*вставить API KEY*)

### Как  импортировать коллекцию:
1. Открыть Postman
2. В меню в левом верхнем углу выбрать: File -> Import...
3. В открывшемся окне вставить ссылку в поле "Paste cURL, Raw text or URL...".
4. Коллекция испортируется в ваш Postman.
